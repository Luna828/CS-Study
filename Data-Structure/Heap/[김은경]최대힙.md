## HEAP 힙 이란?
* 완전 이진 트리의 일종으로, `우선순위 큐`를 위하여 만들어진 자료구조이다
* 여러개의 값들 중에서 최댓값이나 최솟값을 빠르게 찾아내도록 만들어진 자료구조
* 우선순위가 가장 높은 데이터가 제일 앞에 위치
* 부모는 자식보다 우선순위가 더 높은 데이터가 배치됨
* 일종 반정렬 상태 유지
* 중복 값 허용 (이진 탐색 트리는 중복 X)

## 힙의 종류
* 최대 힙 (max heap)
  - 부모 노드 key >= 자식 노드 key 
* 최소 힙 (min heap)
  - 부모 노드 key <= 자식 노드 key
